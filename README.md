# AI Text Summarizer

> Paste any text and get an instant summary with key points, topics, and sentiment analysis.

## Live Demo
https://nzjulien.github.io/ai-text-summarizer

## Features
- 4 summary lengths: very short, short, medium, detailed
- 5 styles: TL;DR, executive, academic, bullet-only, ELI5
- Multi-language output (EN, FR, ES, DE)
- Key points extraction
- Topic tagging
- Sentiment analysis with explanation
- Word count reduction stats
- Export as .txt or copy to clipboard

## Stack
- Vanilla HTML/CSS/JS
- Claude claude-sonnet-4-6 API
- Zero dependencies, zero backend
